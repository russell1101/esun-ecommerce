# 玉山銀行電商平台

簡易電商購物中心平台，功能包含前台會員購物與後台商品管理。

## 技術棧

| 項目 | 版本 |
|------|------|
| Java | 17 |
| Spring Boot | 3.x |
| MySQL | 8.0 |
| Vue | 3 + Vite |
| Docker | Compose v2 |

## 快速啟動

```bash
docker compose up --build
```

啟動後：
- 前台：http://localhost
- API：http://localhost:8080
